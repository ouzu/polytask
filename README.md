NAME:
   polytask - taskwarrior module for polybar

USAGE:
   polytask [global options] command [command options] [arguments...]

COMMANDS:
   next     print the next task
   number   print the number of due tasks
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --hours value  due in the next n hours (default: 12)
   --help, -h     show help (default: false)
